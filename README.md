# 📊 Myntra Sales & Product Performance Analysis

## 📌 Overview  
This project analyzes Myntra product-level sales data using Microsoft Excel to uncover insights related to revenue, product categories, brand performance, and pricing strategies.

The focus was on cleaning raw data, building structured tables, and using pivot-based analysis to derive meaningful business insights.


## 🧰 Tools & Techniques Used  
- Microsoft Excel  
- Pivot Tables & Pivot Charts  
- Slicers for interactive filtering  
- Data Cleaning & Transformation  
- Conditional Formatting  
- Functions: XLOOKUP, INDEX-MATCH, IF (nested)  


## 📂 Dataset Description  

The dataset contains product-level information including:

- Product Name & Brand  
- Product Category  
- Ratings & Rating Count  
- Marked Price & Discounted Price  
- Discount %  
- Revenue per Product  

**Dataset Size:**
- **Total Rows:** 81,365  
- **Total Columns:** 10  

The workflow was divided across multiple sheets:
- Raw Data  
- Cleaning Stages  
- Final Cleaned Dataset  
- Pivot Tables  
- Dashboard (Brand & Category analysis)  


## 🧹 Data Cleaning & Preparation  

- Removed inconsistencies and structured the dataset  
- Created calculated columns:
  - **Discount %**
  - **Revenue**
  - **Total Discount Amount**  
- Ensured correct data types for pricing and numerical fields  
- Built a clean table for analysis  


## 💰 Key Insights & Metrics  

### 💳 Overall Performance  
- **Total Revenue Generated:** ₹15,118,025,004  
- **Total Products Analyzed:** 81,365  


### 🛍️ Category Performance  

**Top 5 Categories by Revenue:**
1. kurta-sets → ₹1,173,946,106  
2. tshirts → ₹999,678,575  
3. dresses → ₹874,391,274  
4. shirts → ₹837,361,547  
5. jeans → ₹749,970,849  

👉 A few categories contribute heavily to overall revenue, with *kurta-sets* leading significantly.


### 🏷️ Brand Performance  

**Top 5 Brands by Revenue:**
1. Roadster → ₹1,352,913,126  
2. HRX by Hrithik Roshan → ₹474,101,059  
3. SASSAFRAS → ₹432,993,884  
4. Philips → ₹361,430,218  
5. HIGHLANDER → ₹332,229,103  

👉 Roadster clearly dominates, contributing significantly more revenue than other brands.


### 💸 Pricing & Discount Insights  

- **Average Discount:** 43.57%  
- **Average Rating:** 4.17  

👉 High discount percentages suggest aggressive pricing strategies to drive sales volume.


### ⭐ Product Quality Insight  

- Average rating across products is **4.17**, indicating generally positive customer feedback  
- Higher-rated products tend to maintain strong sales performance  


## 📊 Dashboard & Visualization  

An interactive Excel dashboard was created using:

- **Pivot Charts** for visualizing category and brand performance  
- **Slicers** to filter data dynamically (e.g., by category or brand)  
- Clean visual layout for quick business insights  

👉 This enables quick exploration of data and supports better decision-making without writing formulas repeatedly.


## 🎯 Business Recommendations  

- Focus inventory and marketing on **top-performing categories like kurta-sets and tshirts**  
- Strengthen partnerships with **high-performing brands like Roadster and HRX**  
- Continue leveraging **discount strategies** to drive volume  
- Promote **high-rated products** to improve conversion rates  


## 📎 Conclusion  

This project demonstrates how Excel can be used to transform raw data into structured insights. By combining data cleaning, calculated fields, pivot analysis, and interactive dashboards, it becomes possible to identify key revenue drivers and product trends.

The analysis shows that revenue is concentrated among a few categories and brands, while pricing strategies such as discounts play a significant role in driving sales.
